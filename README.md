# 如何下载

## 我需要下载全部文件

- 如果您会使用 Git，直接使用 Git 将文件克隆到本地即可。如果有能力，建议使用这种方式，方便获取更新。
- 如果您不会使用 Git，单击页面上方绿色的 Code 按钮，选取 Download ZIP 即可。

## 我只需要下载某一本

- 点击上方文件夹，打开自己想要下载的那一本，并单击 Download。
